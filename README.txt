## Init
You need ruby installed.

    $ gem install sass
    $ npm install
    $ ./node_modules/.bin/bower install
    $ ./node_modules/.bin/gulp 

## Gulp
Gulp has tree avaible tasks
- default: build scripts and styles and watch for modification
- build: build scripts and styles one time
- sync: build scripts and styles and watch for modification and launc browser-sync proxy


## Update dependencies
    $ npm update
    $ bower update
